Maze Navigator ESP32

Description

An intelligent ESP32-powered autonomous robot designed to navigate and solve complex mazes using IR sensors and real-time decision making.

Components Used

- ESP32
- IR Sensors
- L298N Motor Driver
- DC Motors
- Battery

Working

The robot detects paths using IR sensors and follows them by controlling motors. It automatically takes decisions at turns and solves the maze efficiently.
